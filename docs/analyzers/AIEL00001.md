# AIEL00001 - General - Root Dependency Required

## Summary

Every assembly that references Aiel must declare exactly one public root dependency type.

## Problem

An assembly either does not declare a root dependency or declares more than one.

## Solution

Declare exactly one public sealed class with a parameterless constructor that inherits from AielApplication or AielDependency.

## Why This Matters

The analyzer uses that type as the root for configuration, initialization, and dependency resolution.

## Additional Information

- Category: Usage
- Severity: Error
- Enabled by default: Yes
