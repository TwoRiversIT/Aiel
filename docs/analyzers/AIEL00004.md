# AIEL00004 - General - Ambiguous Project Type

## Summary

The generator found more than one project type in the same assembly.

## Problem

An assembly is marked as more than one of WebAssembly, WebApplication, or HostApplication.

## Solution

Make the assembly represent exactly one project type.

## Why This Matters

The generator needs one clear project shape to produce the correct output.

## Additional Information

- Category: Aiel.Generators
- Severity: Error
- Enabled by default: Yes
