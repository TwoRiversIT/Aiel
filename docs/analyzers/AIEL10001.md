# AIEL10001 - Strong ID - Must Be Partial Record Type

## Summary

Strong ID declarations must be partial record types.

## Problem

The type is declared as something other than a partial record struct or partial sealed record.

## Solution

Declare the Strong ID as a partial record struct or partial sealed record.

## Why This Matters

The generator owns the implementation surface and expects to complete the type definition.

## Additional Information

- Category: Aiel.StrongIds.Generators
- Severity: Error
- Enabled by default: Yes
