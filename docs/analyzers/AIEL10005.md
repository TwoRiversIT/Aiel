# AIEL00017 - Strong ID - Must Not Declare Instance Constructors

## Summary

Strong ID declarations must not declare instance constructors.

## Problem

The type defines constructors that conflict with the generated validation and construction flow.

## Solution

Remove instance constructors from the Strong ID declaration.

## Why This Matters

The generator owns all instance construction so it can enforce consistent validation.

## Additional Information

- Category: Aiel.StrongIds.Generators
- Severity: Error
- Enabled by default: Yes
