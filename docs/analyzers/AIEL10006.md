# AIEL00018 - Strong ID - Unsupported Backing Type

## Summary

Strong ID declarations must use a supported backing type.

## Problem

The type uses a backing type other than Guid, int, long, or string.

## Solution

Change the Strong ID backing type to one of the supported types.

## Why This Matters

The generator only knows how to emit code for the supported backing types.

## Additional Information

- Category: Aiel.StrongIds.Generators
- Severity: Error
- Enabled by default: Yes
