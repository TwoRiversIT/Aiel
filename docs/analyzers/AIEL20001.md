# AIEL00006 - Authorization - Action Has No Authorization Story

## Summary

Every IAction implementation must either have an authorization checker or explicitly opt out.

## Problem

The analyzer found an IAction with no concrete IActionAuthorizationChecker and no DoesNotRespectAuthority annotation.

## Solution

Add a checker for the action or mark it with DoesNotRespectAuthority and a non-empty reason.

## Why This Matters

The rule is fail closed, so missing authorization is treated as an error.

## Additional Information

- Category: Authorization
- Severity: Error
- Enabled by default: Yes
