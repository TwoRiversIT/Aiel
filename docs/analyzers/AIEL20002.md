# AIEL20002 - Authorization - DoesNotRespectAuthority Reason Is Empty

## Summary

DoesNotRespectAuthority must include a non-empty reason.

## Problem

The marker was applied with an empty or whitespace reason.

## Solution

Provide an auditable reason that explains why the action is permission free.

## Why This Matters

Empty reasons make the opt out impossible to review later.

## Additional Information

- Category: Authorization
- Severity: Error
- Enabled by default: Yes
