# Aiel.Mediator.Analyzers

This project contains Roslyn analyzers for the Aiel.Mediator library. These analyzers help developers identify potential issues and improve the usage of the Aiel.Mediator library in their code. By analyzing the code, these analyzers can provide insights and suggestions to enhance the implementation of the mediator pattern, ensuring better performance and maintainability in applications that utilize the Aiel.Mediator library.
