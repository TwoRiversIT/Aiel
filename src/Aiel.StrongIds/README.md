# Aiel.StrongIds

`Aiel.StrongIds` provides the runtime contracts and authoring attribute for Strong IDs.

Referencing this package also delivers the Strong ID source generator at compile time so declarations such as `[StrongId<Guid>]` generate their implementation without a separate tooling package reference.
